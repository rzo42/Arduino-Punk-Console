# Arduino-Punk-Console
Pulse width audio for the Arduino Uno R3
Version:  B.2.3.1                                     
By:  Ryan Liston                               
ate: January 2,2023                                    
                                                                   
"Fpot" @ A1 controls note frequency                          
"WPot"  @ A0 comtrols pulse width                           

Parts:                                                          
       1 Aurduino Uno R3                                           
       2 Potentiometers (100k prefered;10k recommended minimum.)   
       1  Piezo Spkrer                                             
                               ...Wires (as needed)
